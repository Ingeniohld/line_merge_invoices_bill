Sale Quotation Multi Confirmation

About this module
  • This module is used to confirm multiple sale orders at the same time.
  • Users can confirm multiple orders in the tree view.
1. Sales
  • Navigate to sales and select multiple orders.
  • Click on the action and then confirm Order(s).
2. Confirm Multiple Order

  • Status change from quotation to sales order.
